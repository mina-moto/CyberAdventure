# CyberAdventure

## Description
Side Scroll Action Game.  
Aim at the goal.

## Requirement
Java8

## Operating
Stage
- ↑:Jump
- ←→:Move
- Z:Attack
- Space:Pauze

Other
- Z:Forward
- X:Back
- Cursor Key:Choice
